# User Manual

## 1. Getting Started
- Navigate to the application URL.
- Use the navigation bar at the top (or hamburger menu on mobile) to switch between Players, Teams, and Matches.

## 2. Managing Players
- Go to the **Players** section.
- Fill in the "Create Player" form and click "Register Player".
- Use the ✏️ icon to edit a player and the 🗑️ icon to delete.

## 3. Managing Teams
- Go to the **Teams** section.
- Enter a Team Name and select a Captain from the registered players.
- Click "Create Team".
- The team table will display the Team Name and the assigned Captain's name.

## 4. Scheduling Matches
- Go to the **Matches** section.
- Select Team 1 and Team 2 from the dropdowns (must be different).
- Choose the Date and Time.
- Enter the Venue/Location.
- Click "Schedule Match".
- The schedule table shows all upcoming matches with their current status.
